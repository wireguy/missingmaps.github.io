    Comment contribuer

    Si vous êtes nouveau contributeur, nous vous recommandons de regarder les vidéos de la section 'Débutants' ci-dessous pour bien démarrer. Si vous avez déjà cartographié suivez le lien 'Cartographier'. Il existe aussi d'autres tutoriels disponible via LearnOSM.

    Débutants

    PLUS DE TUTORIAUX
    YOUTUBE
    Débutants
    1. Créez un Compte OpenStreetMap

    Allez sur le site OpenStreetMap inscrivez-vous vous pour créer un compte. Suivez le lien dans le mail qui vous sera envoyé pour valider la création de votre compte.

    2. Apprenez à cartographier les bâtiments

    Les bâtiments sont un des éléments les plus utiles que vous pouvez ajouter à OpenStreetMap (et c'est amusant aussi)! Apprenez comment ajouter ou éditer les bâtiments dans OSM avec l'éditeur iD.

    3. Apprenez à cartographier les routes

    Les routes sont également une des choses les plus utiles à ajouter à OpenStreetMap! Apprenez à ajouter et éditer des routes dans OSM avec l'éditeur iD.

    4. Apprenez à utiliser le gestionnaire de tâches

    Allez sur le Gestionnaire de Tâches pour trouver un projet et une tâche qui vous intéresse. Apprenez comment bloquer une tâche pour l'éditer, sélectionnez votre outil d'édition et changez la configuration du fond de plan.
